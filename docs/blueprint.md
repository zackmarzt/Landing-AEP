# **App Name**: Bosque Mananciais Landing

## Core Features:

- Hero Section: Eye-catching hero section with school name, logo, and a brief introduction.
- About Us: Information about the school's mission, values, and history.
- Programs Offered: Detailed descriptions of the educational programs and extracurricular activities offered by the school.
- Image Gallery: Showcase of school facilities, events, and student life.
- Testimonials: Student and parent testimonials to build trust and credibility.
- Contact Information: Contact form, address, phone number, and map for easy inquiries.

## Style Guidelines:

- Primary color: Olive green (#B7C727) to reflect the school's branding and create a natural, welcoming feel.
- Background color: Very light olive green (#F0F3DD) - desaturated version of the primary.
- Accent color: A muted yellow green (#96C727), slightly shifted hue to complement the olive green and provide contrast.
- Body font: 'PT Sans', a humanist sans-serif for readability and a modern look.
- Headline font: 'Space Grotesk', a geometric sans-serif for headings and important titles, adding a touch of modernity.
- Use nature-inspired icons to connect with the 'Bosque' theme, using the primary olive green as the dominant color.
- Clean and spacious layout with clear sections for easy navigation and readability.
- Subtle animations on scroll to enhance user engagement and create a smooth browsing experience.
